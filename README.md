# IPL Match Analytics – Wooble Hackathon 2026
## Overview

This project analyzes 5 seasons of Indian Premier League (IPL) ball-by-ball data to uncover match-winning patterns, batting and bowling performance, and the impact of toss decisions.

The analysis was built as part of the Wooble IPL 2026 Hackathon using Python-based data analytics and visualization techniques.

## Objectives

The project answers the following questions:

Do teams that win the toss actually win more matches?
Which phase of the game is most important for winning?
Powerplay
Middle Overs
Death Overs
Who are the top 5 batters and bowlers across 5 IPL seasons?
